# Escritorio Dixital – Guía de widgets

Escritorio Dixital é unha pizarra de traballo con “xanelas” pensada para a aula. Podes abrir os widgets que precises, movelos, redimensionalos e combinalos para crear a túa “mesa de traballo” para unha actividade.

## Como se usa

- **Abrir un widget**: desde a barra inferior ou desde a **Biblioteca de widgets**.
- **Pechar todo**: no menú contextual do escritorio.
- **Perfís**: crea varios escritorios e cambia entre eles.
- **Fondos**: escolle un fondo predefinido ou sobe un propio.

## Catálogo de widgets

### Organización e aula

- **Lista de tarefas**: lista de tarefas/recordatorios. Útil para a axenda do día, pasos dunha práctica, rúbrica rápida ou “plan da sesión”.
- **Control de asistencia**: rexistro rápido de asistencia e estado do grupo.
- **Xerador de grupos**: crea grupos de forma aleatoria ou equilibrada para traballo cooperativo.
- **Calendario**: apoio visual para datas, entregas e planificación.

### Tempo e ritmo

- **Temporizador**: conta atrás para actividades por estacións, exames curtos, traballo por quendas, etc.
- **Cronómetro.
- **Metrónomo**: marca un pulso constante. Útil para música, lectura rítmica, dinámicas de clase ou traballo por intervalos.

### Matemáticas e ciencias

- **Calculadora científica**: cálculos comúns en ciencias e matemáticas.
- **Conversor de unidades para problemas e prácticas.
- **Markdown + LaTeX.

### Pizarra, escritura e recursos

- **Bloc de notas**: notas rápidas para consignas, ideas, exemplos, listas, etc.
- **Pizarra de debuxo.
- **Carrusel de imaxes.
- **Visor web**: abre unha páxina web dentro do escritorio para usar un recurso en liña sen cambiar de pestana.
- **Visor PDF**: abre ficheiros PDF dentro do escritorio.
- **Visor directo**: pensado para mostrar contido web ou recursos externos de forma cómoda durante a explicación.
- **HTML.

### Interacción, azar e xogos educativos

- **Dado**: tiradas rápidas para azar, gamificación ou selección aleatoria.
- **Ruleta aleatoria**: sortea quendas, equipos, preguntas, alumnado ou retos.
- **Xerador de QR**: crea códigos QR para compartir ligazóns ou recursos coa clase.
- **Semáforo**: sinal visual de “silencio / traballo / atención” ou estados de actividade.
- **Medidor de son**: indicador visual do nivel de ruído para a autorregulación do grupo.

### Xogos de lóxica e dinámica de clase

- **Tres en raia**: xogo rápido para repaso, preguntas‑respostas, motivación ou “duelos” de conceptos.
- **Memoria**: xogo de parellas para vocabulario, conceptos, imaxes e definicións.
- **Puzzle deslizante**: quebracabezas para atención, lóxica e momentos de transición.

### Traballo con xestos

- **Xestos de traballo**: ferramenta de apoio para dinámicas de aula baseadas en sinais/xestos e rutinas.

## Ideas de uso na clase

- **Inicio de sesión**: Lista de tarefas + Temporizador + Semáforo.
- **Explicación**: Markdown + LaTeX + Pizarra de debuxo + Visor web.
- **Cooperativo**: Xerador de grupos + Temporizador + Medidor de son.
- **Gamificación**: Ruleta aleatoria + Dado + Marcador de puntos.
