# Idazmahai Digitala – Widgeten gida

Idazmahai Digitala ikasgelarako pentsatutako “leihoekin” lan‑taula bat da. Behar dituzun widgetak ireki, mugitu, tamaina aldatu eta konbina ditzakezu jarduera baterako zure “lan‑mahaia” sortzeko.

## Nola erabiltzen da

- **Widget bat ireki**: beheko barratik edo **Widgeten Liburutegitik**.
- **Dena ixtea**: mahaigaineko testuinguru‑menuan.
- **Profilak**: sortu hainbat mahaigain eta aldatu haien artean.
- **Atzeko planoak**: aukeratu aurrez zehaztutako bat edo igo zurea.

## Widgeten katalogoa

### Antolaketa eta ikasgela

- **Zereginen zerrenda**: zereginen/oharren zerrenda. Erabilgarria eguneko agenda, praktika‑urratsak, errubrika azkarra edo “saio‑plana” egiteko.
- **Asistentzia**: asistentzia eta talde‑egoera erregistro azkarra.
- **Talde‑sortzailea**: taldeak ausaz edo orekatuta sortu lan kooperatiborako.
- **Egutegia**: datak, entregak eta plangintza ikusteko laguntza bisuala.

### Denbora eta erritmoa

- **Kronometroa**: atzerako kontua estazio‑lanetarako, proba laburretarako, txandakako lanerako, etab.
- **Kronometroa.
- **Metronomoa**: pultsua markatzen du. Musika, irakurketa erritmikoa, ikasgelako dinamikak edo tarte‑lana egiteko.

### Matematika eta zientziak

- **Kalkulagailu zientifikoa**: matematika eta zientzietako kalkulu arruntak.
- **Unitate‑bihurgailua ariketetarako eta praktiketan.
- **Markdown + LaTeX.

### Arbel, idazketa eta baliabideak

- **Ohar‑blokea.
- **Marrazteko arbela.
- **Irudi‑karusela.
- **Web‑bisorea**: web orri bat mahaigain barruan ireki, fitxaz aldatu gabe.
- **PDF ikustailea**: PDF fitxategiak mahaigainean ireki.
- **Zuzeneko bisorea**: azalpenetan web edukia edo kanpoko baliabideak eroso erakusteko.
- **HTML.

### Interakzioa, ausazkotasuna eta joko didaktikoak

- **Dadoa**: ausazko tiradak, gamifikazioa edo hautaketa azkarrerako.
- **Zorizko erruleta**: txandak, taldeak, galderak, ikasleak edo erronkak aukeratzen ditu.
- **QR sortzailea**: QR kodeak sortu estekak edo baliabideak partekatzeko.
- **Semaforoa**: “isiltasuna / lana / arreta” edo jarduera‑egoerak adierazteko seinale bisuala.
- **Soinu‑neurgailua**: zarata maila erakusten du taldearen auto‑erregulaziorako.

### Logika‑jokoak eta gelako dinamikak

- **Hiru lerroan**: joko azkarra errepasorako, galdera‑erantzunetarako, motibaziorako edo kontzeptu‑dueluetarako.
- **Memoria**: bikoteak aurkitzeko jokoan, hiztegia, kontzeptuak, irudiak eta definizioak lantzeko.
- **Puzzle irristakorra**: arreta, logika eta trantsizioetarako.

### Keinuekin lana

- **Lan‑keinuak**: seinale/keinuetan oinarritutako ikasgelako errutinak laguntzeko tresna.

## Ikasgelarako ideiak

- **Saioaren hasiera**: Zereginen zerrenda + Temporizadorea + Semaforoa.
- **Azalpena**: Markdown + LaTeX + Marrazteko arbela + Web‑bisorea.
- **Kooperatiboa**: Talde-sortzailea + Temporizadorea + Soinu-neurgailua.
- **Gamifikazioa**: Zorizko erruleta + Dadoa + Puntu-markagailua.
