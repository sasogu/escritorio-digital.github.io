# Escritorio Digital – Guía de widgets

Escritorio Digital es una pizarra de trabajo con “ventanas” pensada para el aula. Puedes abrir los widgets que necesites, moverlos, redimensionarlos y combinarlos para crear tu “mesa de trabajo” para una actividad.

## Cómo se usa

- **Abrir un widget**: desde la barra inferior o desde la **Librería de widgets**.
- **Cerrar todo**: en el menú contextual del escritorio.
- **Perfiles**: crea varios escritorios y cambia entre ellos.
- **Fondos**: elige un fondo predefinido o sube uno propio.

## Catálogo de widgets

### Organización y aula

- **Lista de tareas**: lista de tareas/recordatorios. Útil para agenda del día, pasos de una práctica, rúbrica rápida o “plan de la sesión”.
- **Control de asistencia**: registro rápido de asistencia y estado del grupo.
- **Generador de grupos**: crea grupos de forma aleatoria o equilibrada para trabajo cooperativo.
- **Calendario**: apoyo visual para fechas, entregas y planificación.

### Tiempo y ritmo

- **Temporizador**: cuenta atrás para actividades por estaciones, exámenes cortos, trabajo por turnos, etc.
- **Cronómetro.
- **Metrónomo**: marca un pulso constante. Útil para música, lectura rítmica, dinámicas de clase o trabajo por intervalos.

### Matemáticas y ciencias

- **Calculadora científica**: cálculos comunes en ciencias y matemáticas.
- **Conversor de unidades para problemas y prácticas.
- **Markdown + LaTeX.

### Pizarra, escritura y recursos

- **Bloc de notas**: notas rápidas para consignas, ideas, ejemplos, listas, etc.
- **Pizarra de dibujo.
- **Carrusel de imágenes.
- **Visor web**: abre una página web dentro del escritorio para usar un recurso online sin cambiar de pestaña.
- **Visor PDF**: abre archivos PDF dentro del escritorio.
- **Visor directo**: pensado para mostrar contenido web o recursos externos de forma cómoda durante la explicación.
- **HTML.

### Interacción, azar y juegos educativos

- **Dado**: tiradas rápidas para azar, gamificación o selección aleatoria.
- **Ruleta aleatoria**: sortea turnos, equipos, preguntas, alumnos/as o retos.
- **Generador de QR**: crea códigos QR para compartir enlaces o recursos a la clase.
- **Semáforo**: señal visual de “silencio / trabajo / atención” o estados de actividad.
- **Medidor de sonido**: indicador visual del nivel de ruido para autorregulación del grupo.

### Juegos de lógica y dinámica de clase

- **Tres en raya**: juego rápido para repaso, preguntas-respuestas, motivación o “duelos” de conceptos.
- **Memoria**: juego de parejas para vocabulario, conceptos, imágenes y definiciones.
- **Puzzle deslizante**: rompecabezas para atención, lógica y momentos de transición.

### Trabajo con gestos

- **Gestos de trabajo**: herramienta de apoyo para dinámicas de aula basadas en señales/gestos y rutinas.

## Ideas de uso en clase

- **Inicio de sesión**: Lista de tareas + Temporizador + Semáforo.
- **Explicación**: Markdown + LaTeX + Pizarra de dibujo + Visor web.
- **Cooperativo**: Generador de grupos + Temporizador + Medidor de sonido.
- **Gamificación**: Ruleta aleatoria + Dado + Marcador de puntos.
